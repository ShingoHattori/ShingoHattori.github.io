Contents on this site

  • 1D Physics Simulator : In one dimension, numerous perfectly elastic balls are subjected to random forces (thermal fluctuations). Due to this, the balls collide with their neighbors and reflect, resulting in a back-and-forth motion within a specific range. When placed on a slope, in two or three dimensions, the balls tend to accumulate at the bottom due to the energy inefficiency of moving uphill. As a result, fewer balls are observed at higher altitudes, demonstrating an exponential decline known as "thermal activation" or the "Boltzmann distribution." However, in a one-dimensional scenario, the balls cannot slip past each other, and it's hypothesized that the average distance (or density) between the balls remains constant regardless of their position on the slope. This simulator was developed to validate this hypothesis.
  • UniChar View : A web tool for seamless conversion between Unicode characters, code points, and UTF-8 byte sequences. Input a character to see its Unicode code point and UTF-8 byte representation, or enter code points/bytes to reconstruct the original characters. Supports batch processing of multiple characters, making it useful for character set analysis, encoding verification, and debugging character encoding issues. Built with developers and digital text processors in mind, this tool provides intuitive visualization of how characters are represented in different digital formats.
